Password Method

The main concept of this password protect page is to open a new window that links visitor to the correct main page upon entering the correct password. Password input form was created in HTML, where the form input is processed by Javascript to compare with the developer password. If the user/visitor password matches with the developer password, submitting password should bring the user to Winnie the Pooh page.
